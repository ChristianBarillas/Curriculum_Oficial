QuickSort

It does not require extra store! Suppose we have the following sequence: [ 2, 0, 7, 4, 3 ] We choose 3 (last element) as the pivot. After doing 4 comparisons we get the two halves: [ 2, 0 ] (3) [ 7, 4 ] Now, the same process repeats for the two halves. We choose (0) as a pivot for the lesser half, and (4) for the greater half. After a comparison for each half, we get: [ (0) [2] ] (3) [ (4) [7] ] Which is a sorted sequence.
